Project Introduction

Treasure Map for Study Abroad is a planning and application assistance system designed specifically for Xi'an Jiaotong-Liverpool University (XJTLU) students.

Project Background

With globalization, more Chinese students choose to study abroad. However, the application process is complex, involving school selection, material preparation, language exams, and submission. Many students feel lost and stressed without effective tools.

Core Objectives

  • Provide an integrated study abroad planning platform
  • Offer personalized recommendations
  • Track application progress effectively
  • Provide emotional support during the process

User Needs Summary

  • Information Access: Global university info, application requirements
  • Planning Management: Progress tracking, task management
  • Emotional Support: Stress relief, psychological support
  • Intelligent Assistance: Personalized suggestions, AI help

Core Features

Interactive Map

Intuitively display global university distribution. Click markers to view details including rankings, majors, and requirements.

Progress Tracking

Visual progress bars track application steps: language exams, materials preparation, and online submissions.

Emotional Support

Mood tracking with inspirational quotes and stress relief methods. Gamification elements boost confidence.

AI Assistant

AI-powered chat providing personalized study abroad advice, success rate analysis, and essay guidance.

Feature Highlights

  • Gamified Design: Achievement systems increase engagement
  • Personalized Experience: Customized suggestions based on user data
  • Data Visualization: Intuitive progress and university info display
  • Multi-dimensional Support: Comprehensive support from info to emotions

Design Process

Design Methodology

1

User Research

Questionnaires and interviews with XJTLU students to understand application challenges.

2

Requirements Analysis

Organize feedback to identify core requirements and functional priorities.

3

Prototype Design

From low-fidelity sketches to high-fidelity prototypes.

4

User Testing

Invite users to test prototypes and collect feedback.

5

System Implementation

Implement complete functions with frontend technology.

Technical Architecture

  • Frontend: HTML5, CSS3, JavaScript
  • Map: Leaflet.js
  • Storage: localStorage
  • AI: DeepSeek API
  • Style: CSS Variables, Responsive Design
