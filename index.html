Project Introduction

Treasure Map for Study Abroad is a study abroad planning and application assistance system designed specifically for Xi'an Jiaotong-Liverpool University (XJTLU) students. The project aims to provide students with a comprehensive study abroad planning experience by integrating features such as map exploration, AI assistance, progress management, and emotional support.

Project Background

With the development of globalization, more and more Chinese students choose to study abroad. However, the application process is complex and tedious, involving multiple steps such as school selection, material preparation, language exams, and application submission. Many students feel lost and stressed during this process, lacking effective planning tools and support systems.

Problem Statement

  • Students find it difficult to intuitively understand the distribution and detailed information of global universities
  • The application process is complex and lacks clear progress tracking mechanisms
  • Students face huge pressure during the application process and lack emotional support
  • Lack of personalized application suggestions and data processing tools

Core Objectives

This project aims to provide students with an integrated and personalized study abroad planning platform through innovative design and technical implementation, helping them complete the application process more efficiently and confidently.

User Needs

  • Information Access: Global university information, application requirements, rankings, etc.
  • Planning Management: Application process tracking, task management, time planning
  • Emotional Support: Stress relief, emotional management, psychological support
  • Intelligent Assistance: Personalized suggestions, material preparation guidance, data analysis

Core Features

Convenient School Preview

Intuitively display the distribution of global universities through an interactive world map. Click on university markers to view detailed information, including rankings, major settings, application requirements, etc. Supports university collection function, making it convenient for students to manage target institutions.

Clear Application Process Progress

Provide a to-do list for the application process, including key steps such as language exams, material preparation, and online application submission. Track application progress in real-time through visual progress bars, helping students arrange time reasonably and avoid missing important links.

Emotional Support

Built-in mood tracking and emotional management tools, providing inspirational quotes and stress relief methods. Through gamification elements and achievement systems, enhance students' self-confidence and motivation, alleviating anxiety and stress during the application process.

AI-Assisted Data Processing

Integrated AI chat assistant providing personalized study abroad advice and Q&A. Based on students' achievements and background, analyze application success rates, recommend suitable institutions and majors. Assist in processing application materials and provide essay writing guidance.

Feature Highlights

  • Gamified Design: Increase learning fun through achievement systems and user-added achievements
  • Personalized Experience: Provide customized suggestions based on user data
  • Data Visualization: Intuitively display application progress and university information
  • Multi-dimensional Support: Comprehensive support from information, planning to emotions

Design Process

Design Methodology

This project adopts a user-centered design approach, implementing product design through the following steps:

1

User Research

Through questionnaires and interviews, understand the challenges and needs faced by XJTLU students during the application process.

2

Requirements Analysis

Organize user feedback, identify core requirements and functional priorities.

3

Prototype Design

From low-fidelity sketches to high-fidelity prototypes, gradually improve interface design and interaction flow.

4

User Testing

Invite target users to test the prototype, collect feedback and perform iterative optimization.

5

System Implementation

Implement complete functions based on the frontend technology stack, ensuring system stability and user experience.

Technical Architecture

  • Frontend Technology: HTML5, CSS3, JavaScript
  • Map Function: Leaflet.js
  • Data Storage: localStorage
  • AI Integration: DeepSeek API
  • Style Design: CSS Variables, Gradient Effects, Responsive Design
